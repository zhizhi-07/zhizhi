# 更新日志

## 2025-10-23

### 新增功能
- ✅ 字体自定义功能：支持上传字体链接
  - 用户可以添加自己的字体URL
  - 支持 .woff2, .woff, .ttf, .otf 格式
  - 实时预览字体效果
  - 字体管理（删除、应用）
- ✅ 音乐播放器小程序入口（开发中）

### 优化
- ✅ 字体全局应用：所有页面文字都使用自定义字体
- ✅ 字体加载速度优化：
  - 并行加载，速度提升 3-4 倍
  - 智能缓存机制，避免重复加载
  - 应用启动时自动加载字体
  - 显示加载状态提示
- ✅ 导航栏时间背景缩小：更精致的显示效果
- ✅ 通话记录时间样式优化：更紧凑的布局

### 技术改进
- ✅ 使用 FontFace API 动态加载字体
- ✅ CSS 变量全局字体管理
- ✅ 字体缓存机制（useRef + Set）
- ✅ Promise.all 并行加载优化
